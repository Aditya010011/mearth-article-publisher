# Project Moon Hut - Mearth Content Platform

This repository contains the Mearth Article Publisher and related documentation for the Project Moon Hut content platform.

## 🚀 Quick Start

### Local Development

```bash
cd mearth-article-publisher
npm install
npm start
```

Access the application at `http://localhost:3000`

## 📁 Project Structure

- **mearth-article-publisher/** - Main Node.js/Express application for article publishing
- **data-files/** - Configuration and data files
- **documentation/** - Project specifications and guides

## 🎯 Features

- Upload and process Word/PDF articles
- Auto-generate static HTML pages
- Tagging and classification system
- Batch upload capabilities
- Export articles as static site
